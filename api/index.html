ShrutiLekhan API Documentation

The ShrutiLekhan API allows you to integrate high-accuracy Gujarati Speech-to-Text capabilities directly into your own applications. This page outlines the available endpoints, request formats, and provides usage examples.

Base URL

https://rudrakalariya-shrutilekhan-backend.hf.space

Authentication

No authentication required(beta).Please use responsibly.

Content Type

multipart/form-data

Endpoints

  • POST /transcribe - Upload an audio file for transcription

POST /transcribe

Beta

Sends an audio file to the ShrutiLekhan model and returns the transcribed Gujarati text.

Request

POST /transcribe
Body (multipart/form-data)

Field Type Required Description
audio file Yes The audio file to be transcribed. Recommended format is WAV.

Example request (curl)

curl -X POST \ https://rudrakalariya-shrutilekhan-backend.hf.space/transcribe \ -H "Content-Type: multipart/form-data" \ -F "audio=@recording.wav"

Example response

{ "transcription": "નમસ્તે, તમે કેમ છો?" }
