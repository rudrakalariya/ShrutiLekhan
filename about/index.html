About the Project

ShrutiLekhan is an advanced, is a highly optimized Gujarati Automatic Speech Recognition (ASR) system designed to accurately transcribe complex regional audio. It utilizes a fine-tuned Whisper sequence-to-sequence model, specifically adapted to map the nuanced phonetics of the Gujarati language into accurate text.

This project addresses the unique linguistic challenges of Indic scripts, specifically focusing on the complexities of Matras, Anusvaras, and Unicode standardization in machine learning evaluation.

The system features:

  • A fine-tuned acoustic model achieving an exceptional 0.09 validation loss and a highly precise 10.03% Character Error Rate (CER) on the crowdsourced Vistaar Kathbath dataset.
  • An automated text-normalization pipeline leveraging IndicNLP to autonomously resolve Unicode mismatches, strip formatting artifacts, and standardize complex spelling variations.
  • A high-throughput cloud inference pipeline utilizing 16-bit precision (fp16) and batched processing to accelerate large-scale dataset benchmarking.
  • A detailed analytical framework that highlights the limitations of standard Word Error Rate (WER) metrics for agglutinative Indic languages and establishes CER as a more accurate reflection of acoustic mapping.
